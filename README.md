## Overview

This project is an AI-based trading system that predicts stock price movement using machine learning and technical indicators.

It generates buy/sell signals and evaluates performance using backtesting. The system is also integrated with a Streamlit dashboard for visualization.

## Features

* Machine Learning model (Random Forest)
* Technical indicators (RSI, MACD, SMA, EMA, Momentum)
* Buy/Sell signal generation
* Backtesting with profit/loss calculation
* Streamlit dashboard for visualization
* Candlestick chart with signals

## Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* TA (technical indicators)
* Streamlit
* mplfinance

## 💻 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Results

* Model Accuracy: ~53%
* Profitable backtesting with controlled trades
* Realistic trading logic with risk awareness
